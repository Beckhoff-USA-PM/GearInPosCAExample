# GearInPos_CA Example

Simple example project showing GearInPos using Xts. The complication specifically is centered around how various modulo axes can cause the axis to move to unexpected positions and sometimes you need to unwrap the position to operate in absolute.

# To Use
- Set to State 10 to start a "Modulo" coupling
- Set to State 50 to start a "Non modulo" coupling
- set to State 100 to see a reversing sequence
